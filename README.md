## Project Overview
This website was developed for Albany Bread to promote its products and services online. The website provides information about the company, its products, contact details, enquiries, and an image gallery.

## Website Features
- Responsive website design
- Navigation menu
- Search functionality
- Image gallery with Lightbox
- Interactive Google Maps
- Contact form
- Enquiry form validation
- Dynamic date and time display
- SEO optimisation
- Mobile-friendly layout

## Pages
### Home Page (index.html)
- Welcome message
- Product search feature
- Featured products
- Bread gallery

### About Us (about_us.html)
- Company history
- Mission and vision

### Services (services.html)
- Product and service information
- Bread gallery

### Contact Page (contact.html)
- Contact form
- Interactive Google Map

### Enquiries Page (enquiries.html)
- Enquiry form
- Validation and user feedback

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Lightbox2
- Google Maps
